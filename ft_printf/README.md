## Instalation

```shell
$ make
```
## Useage example

```
#include "ft_printf.h"

int main()
{
	ft_printf("I am %d years old", 20);
	return (0);
}
```

```csh
$ cc main.c libftprintf.a -o test.out
$ ./test.out
```
